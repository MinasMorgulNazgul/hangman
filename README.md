# Игра "Виселица"

Консольная версия игры "Виселица". Пользователь должен слово, отгадывая по одной букве. Кол-во букв в загаданном слове известно.

##### Как запустить:

1. Клонируйте этот репозиторий
```
gh repo clone MinasMorgulNazgul/hangman
```
2. Зайдите в папку с игрой
3. Запустите файл main.rb 
```
ruby main.rb
```


##### Как добавлять новые слова в игру:

1. Зайдите в файл data/words.txt
2. Добавьте новое слово с новой строки

На одной строке должно находиться только одно слово.
